# 串流平台 Session 管理器 (Streaming Session Manager) v2.2

這是一個 Chrome 擴充功能，用於匯出與匯入多個串流平台的 Cookies 與 Storage 資料，並支援同平台多個帳號（分身）快速切換。

## 支援平台
- Disney+
- Netflix
- HBO Max / Max
- Amazon Prime Video

## 功能特色
- **多語系支援**：支援繁體中文、簡體中文、英文與日文。
- **多帳號管理**：支援儲存多組 Session 並隨時切換，不需要手動匯出匯入。
- **全平台自動偵測**：匯入全平台 JSON 時，系統會自動辨識並詢問是否儲存為多個獨立分身。
- **專屬管理頁面**：獨立的網頁介面，支援更名、刪除、單獨匯出分身資料。
- **平台篩選**：匯出或匯入時，可自由勾選想要處理的平台。
- **資料完整**：包含 Cookie 與 LocalStorage / SessionStorage。

## 使用方法
### 帳號儲存與切換
1. 開啟目標串流平台頁面並登入。
2. 點擊擴充功能圖示，在「快速儲存目前分身」輸入名稱並點擊「儲存」。
3. 儲存後，下方會出現快捷按鈕，一鍵即可「切換」分身。

### 管理與維護
1. 點擊彈窗上方的「⚙️ 開啟帳號管理分頁」。
2. 在管理頁面中，您可以對分身進行 **更名**、**刪除** 或 **單獨匯出**。

### 舊版 JSON 匯入
- 支援 v2.0 格式。在「匯入 Session」貼上資料，系統會自動偵測是否需要轉換為分身。

---
*註：Storage 資料還原通常需要在該平台的網頁分頁中執行才有效。*
